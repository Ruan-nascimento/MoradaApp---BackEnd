# Utils / Helpers

A pasta `utils` (ou `helpers`) contém funções auxiliares que podem ser usadas em diversas partes do sistema.

**Regras:**
- Use para lógicas genéricas (ex: gerador de ID, formatador de datas, utilitários matemáticos).
- Funções aqui devem ser puras e simples, não devem depender do contexto do Express nem conhecer o banco de dados.
