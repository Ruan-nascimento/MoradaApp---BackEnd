# Models / Entities

A pasta `models` (ou `entities` dependendo da arquitetura e ORM escolhido) representa a estrutura dos seus dados no banco.

**Regras:**
- Aqui ficam as classes ou schemas que mapeiam as tabelas ou coleções do banco (ex: Schemas do Mongoose ou Models do Prisma/TypeORM).
- Geralmente contém a tipagem de dados da entidade central.
